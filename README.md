The application downloads web page to your computer, and allows you to view them at any time.

## Setup

```sh
$ git clone git@github.com:h0x0d9/project-lvl3-s322.git
$ cd project-lvl3-s322
$ make install
$ make build
$ npm link
```

## Run

```sh
$ page-loader --help
```
